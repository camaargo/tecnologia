O que é Python?
É uma linguagem de programação.
O que é uma variável?
Uma variável é um espaço na memória que armazena dados que podem mudar.
O que é um algoritmo?
Um algoritmo é uma sequência de instruções passo a passo para resolver um problema.
O que é uma função?
Uma função é um bloco de código que executa uma tarefa específica e pode ser reutilizado.
Como você cria uma função que soma dois números em JavaScript?
Você cria uma função que soma dois números em JavaScript usando const soma = (a, b) => a + b;.
Como você lê uma linha de entrada do usuário em C++?
Você lê uma linha de entrada do usuário em C++ usando std::cin >> entrada;.
Como você define uma classe simples em Python?
Você define uma classe simples em Python usando class MinhaClasse: pass.
Como você inicializa um array de inteiros em C?
Você inicializa um array de inteiros em C usando int meu_array[5] = {1, 2, 3, 4, 5};.
